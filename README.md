## About The Project

This is my first front-end project using React.js. This project creates a simple PST clock that works real time.

* [![React][React.js]][React-url]


<!-- GETTING STARTED -->
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/PstClock.git
   ```
2. Start NPM project
   ```sh
   npm start
   ```
